Corona Virus Data

Data science project / Jonathan Katz

My project

This projects provides data on the number of confirmed Corona virus (COVID-19) cases, death and recovery cases during the initial period of the Corona virus outbreak. This project provides initial data statistics for whoever is seeking such data.

My sources

The information about COVID-19 in this project is taken from the official World Health Organization website. The pictures are taken from various websites using google search for Corona Virus. Data source: Humanitarian Data Exchange

About COVID-19

COVID-19 is a new strain of coronavirus that has not been previously identified in humans. Common signs of infection include fever, cough, and breathing difficulties. In severe cases, infection causes acute respiratory syndrome and even death.
